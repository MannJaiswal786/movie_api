myFlix movie API(Mann Jaiswal)

Documentation:

This API provides users with access to information about different movies, directors, and genres. It also interacts with a database that stores data about different movies.
The API is built on Node.js and Express application.

Request URL HTTP Method Request body data format Response body data format
Return a list of all the movies /movies GET None A JSON object holding data about all the movies
Example:
[ { "Genre": { "Name": "Genre Name", "Description": "Genre Description" }, "Director": { "Name": "Director's Name", "Bio": "Director's Bio", "Birth": "Date" }, "_id": "ID", "Title": "Movie Title", "Description": "Movie Description", "ImagePath": "movieImg.jpg", "Featured": boolean }]
Return data about a single movie by title to the user /movies/[title] GET None A JSON object holding data about a single movie, containing a description, genre, dirctor
Return data about a genre(description) by title /genre/[name] GET None A JSON object holding data about a genre
Example: [ { "_id": "6117add3c32d54013b3f5a23", "Name": "Thriller", "Description": "Thriller films are known to promote intense excitement, suspense, and a high level of anticipation with nerve-wracking tension." }]
Return data about a director /directors/[name] GET None A JSON object holding a single director's data
Example:
{ "_id": "6117b53a646889ab609c5855", "Name": "Martin Scorsese", "Bio": "One of the major figures of the new Hollywood era, he is widely regarded as one of the most significant and influential directors in the film history", "Birth": "1942-11-17T00:00:00.000Z" }
Allow new users to register /users POST A JSON object holding data about a user to add:
{
"Username": "String",
"Password": "String",
"Email": "String",
"Born": "Date",
} 		Response:
{
"id": "3"
"Username": "mannjaiswal",
"Password": "careerfoundry",
"Email": "mannjaiswal5@gmail.com",
"Born": "1999-09-11"}
Allow users to update their username /users/[username] PUT Send query parameter to{myinfo:[ { username:String, email:String, password:String, birthday: Date, FavoriteMovies: [] } ]} Response:
{ "FavoriteMovies": [ "6112fa959c6dacfcc543e59b" ], "_id": "6113453daf5d66f1c029f460", "Username": "Bruce Wayne", "Password": "Black", "Email": "Batman@gmail.com", "Born": "1975-05-05T00:00:00.000Z" }
Allow users to add a movie to their favorites list /users/[username]/movies/[movieID] POST Send data through URL adding the username and the movieID in the requested parameters
example:
/users/Bruce%20Wayne/movies/29u8329v93i0952009r4 		A JSON object holding data about the movie was added to your favorites list
example:
{ "FavoriteMovies": [ "6112fa959c6dacfcc543e59b" ]}
Allow users to remove a movie from their favorites list /users/[username]/movies/[movieID] DELETE Send data through URL adding the username and the movieID in the requested parameters
example:
/users/Bruce%20Wayne/movies/29u8329v93i0952009r4 		A text message indicating that the movie was removed from the favorites list
Allow existing users to deregister /users/[username] DELETE Send data through URL in the requested parameters
example:
/users/Bruce%20Wayne 		A text message indicating that the user has been deregistered
