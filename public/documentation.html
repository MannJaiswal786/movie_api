myFlix movie API(Mann Jaiswal)

Documentation:

This API provides users with access to information about different movies, directors, and genres. It also interacts with a database that stores data about different movies.
The API is built on Node.js and Express application.

Request URL HTTP Method Request body data format Response body data format
Return a list of all the movies /movies GET None A JSON object holding data about all the movies
Return data about a single movie by title to the user /movies/[title] GET None A JSON object holding data about a single movie, containing a description, genre, dirctor
Return data about a genre(description) by title /movies/genre/[Name] GET None A JSON object holding data about a genre
Return data about a director /movies/directors/[Name] GET None A JSON object holding a single director's data
Allow new users to register /users POST A JSON object holding data about a user to add:

Allow users to update their username /users/[username] PUT None A text message indicating the user's username was updated
Allow users to add a movie to their favorites list /users/[username]/favorites/[movieID] POST None A JSON object holding data about the movie was added to your favorites list
Allow users to remove a movie from their favorites list /users/[username]/favories/[movieID] DELETE None A text message indicating that the movie was removed from the favorites list
Allow existing users to deregister /user/[username]/deregister DELETE None A text message indicating that the user has been deregistered
